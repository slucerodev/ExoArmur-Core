# ExoLock (ExoArmur ExoLock)

ExoLock defines the Stage 1 contract surface for deterministic incident
**time-locking** via the Temporal Authority Envelope (TAE).

- **Stage**: V2 Stage 1 (contracts only, inert).
- **Runtime**: No executable code or tests in this module.
- **Boundary**: Core remains independent; ExoLock is exportable and optional.

Contracts are defined in `contracts/tae_v1.md` with supporting documentation in
`docs/TAE_OVERVIEW.md`.
