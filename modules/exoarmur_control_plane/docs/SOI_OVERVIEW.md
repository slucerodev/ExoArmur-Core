# Signed Operator Intent (SOI) — Overview

**Stage**: V2 Stage 1 (contracts only, inert)

The Control Plane module defines the contract surface for **Signed Operator
Intent (SOI)**, arbitration records, and emergency kill authority semantics.
This module provides **no runtime behavior**. It establishes the normative
language for verifying operator intent without executing it.

## Scope
- Define what a Signed Operator Intent is and the minimum required fields.
- Define arbitration record semantics and admissible evidence for disputes.
- Define emergency kill authority semantics and recording requirements.

## Out of Scope (Stage 1)
- No execution or orchestration of operator intents.
- No policy evaluation or automated approvals.
- No runtime enforcement or network wiring.
